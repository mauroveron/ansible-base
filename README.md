Ansible Base
============

Base software and configuration for Debian systems.

Requirements
------------

None

Role Variables
--------------

- `timezone`: The timezone
- `locale`: Locale string (e.g.: `en_NZ`)

Dependencies
------------

None

Example Playbook
----------------

    - hosts: servers
      roles:
         - { role: maurov.baseline }
